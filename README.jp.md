## 🎯 プロジェクトビジョン
**Flux AI Terminal** は単なるエミュレータではありません。移動中の開発者を支援するためにゼロから構築されたネイティブ実行環境です。**Rust** のパフォーマンスと**ローカル AI** のインテリジェンスを活用することで、Flux はポケットに収まる、低遅延で安全かつ拡張性の高い Linux ワークステーションを提供します。

### 🌟 なぜ Flux なのか？
1. **モバイル上のデスクトップパワー:** コンパイラ、ビルドツール、Web サーバーをネイティブに実行。
2. **オフラインインテリジェンス:** インターネット接続なしで AI を実行し、プライバシーを保護。
3. **強化されたセキュリティ:** ソースコードは生体認証暗号化とリアルタイムコマンドファイアウォールによって保護されます。

## 🏗️ アーキテクチャの深掘り

### 🦀 Rust 核心エンジン
Flux の核心は、Rust で実装された非同期・非ブロッキングカーネルです。`tokio` ランタイムを利用して、最小限の CPU オーバーヘッドで数千の同時タスクを管理します。

#### コンポーネント:
- **PTY エミュレーション:** NeoVim や htop などの複雑な TUI アプリをレンダリングするための、完全な Xterm 互換疑似ターミナル。
- **VFS (仮想ファイルシステム):** ホストシステムを修正することなく、完全な Ubuntu rootfs を提供する OverlayFS スタイルのサンドボックス。
- **パッケージマネージャ:** アトミックなパッケージ管理のための `dpkg` および `apt` のネイティブ実装。

### 🧠 AI RAG エンジン
Flux にはローカライズされた **検索拡張生成 (RAG)** エンジンが含まれています。ローカルのドキュメントや man ページをベクトルデータベースにインデックス化し、AI がオフラインで文脈に沿った正確な提案を提供できるようにします。

## 🛡️ セキュリティホワイトペーパー

### 1. 暗号化エンクレーブ
すべての機密トークンと git 認証情報は、**AES-256-GCM** 暗号化された保管庫に保存されます。キーは、デバイスのハードウェアエンクレーブ (Secure Element) との生体認証ハンドシェイクに成功した後にのみ派生されます。

### 2. 正規表現コマンドファイアウォール
すべての入力文字列はヒューリスティックファイアウォールによって監査されます。`rm -rf /` や不正なネットワークアクセスなどの破壊的なパターンが検出されると、コマンドは遮断され、ログに記録されます。
