# 🎨 Flux AI Design System
## "Neon Cyberpunk Minimalism"

Flux AI Terminal uses a unique design language that combines the raw power of the terminal with the sleek aesthetics of modern mobile interfaces.

### 🌈 Color Palette
| Token | Hex | Usage |
| :--- | :--- | :--- |
| **Background** | `#0A0E14` | Primary app background |
| **Surface** | `#01060E` | Terminal and card surfaces |
| **Neon Green** | `#00FF9F` | Success states, accents, primary buttons |
| **Neon Blue** | `#00B8FF` | Info states, AI interactions |
| **Neon Red** | `#EA6C73` | Error states, dangerous command warnings |

### 🖋️ Typography
- **Primary Font:** Inter (Modern Sans)
- **Monospace Font:** JetBrains Mono (For terminal and code)

### ✨ Visual Effects
- **Glassmorphism:** Navigation drawers and overlays use 15% opacity backgrounds with 20px blur.
- **Haptic Feedback:** Every terminal keystroke and command success triggers subtle haptic patterns.
- **Micro-Animations:** Terminal output uses a "Typewriter" fade-in effect for a more premium feel.

### 🛡️ Security UI
- **Locked State:** Darkened background with a pulsing neon green fingerprint icon.
- **Firewall Intervention:** A bold red border flash around the terminal when a command is blocked.
