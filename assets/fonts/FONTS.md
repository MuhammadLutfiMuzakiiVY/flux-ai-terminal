# Flux Fonts Bundle Strategy

This directory contains the core typography assets for the Flux AI Terminal cross-platform UI.

## Minimal Bundle Architecture
Following the ideal configuration for a lightweight, fast developer environment:

```text
assets/fonts/
├── JetBrainsMono-Regular.ttf
├── JetBrainsMono-Bold.ttf
├── FiraCode-Regular.ttf
├── CascadiaCode-Regular.ttf
├── UbuntuMono-Regular.ttf
├── Hack-Regular.ttf
├── Inter-Regular.ttf
├── Inter-Bold.ttf
├── Orbitron-Bold.ttf
└── MesloLGSNF-Regular.ttf
```

## Categories

1. **Terminal Fonts (Monospace)**
   - JetBrains Mono (Primary/Default)
   - Fira Code
   - Cascadia Code
   - Ubuntu Mono
   - Hack

2. **Terminal Fonts (Nerd Fonts)**
   - MesloLGS Nerd Font (For Powerline/Zsh icons)

3. **UI Fonts (Sans Serif)**
   - Inter (Sidebar, Settings, File Explorer)

4. **Display Fonts (Branding/Logo)**
   - Orbitron (Splash screen, banners)
